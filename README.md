NG-Jakkel - Angular Jakkel Implementation
======================================

NG-Jakkel is a Angular Jakkel (ACL) Implementation

### Installation
---
```sh
$ bower install --save ng-jakkel
```

### Api
```javascript
angular.module('MyApp', ['jakkel']);
```
